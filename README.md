A **RESTful** backend service for an **E-commerce application** built using **Spring Boot**.
This project provides APIs for managing products across multiple categories and serves product images locally without using any external cloud or server.

This **backend application:**
  Manages products such as Mobiles, Laptops, Cars, Electronics, Toys, Fashion
  Uses **H2 in-memory database for quick setup and testing**
  Preloads sample data using data.sql
  Serves product images directly from the application using Spring Boot static resources
  Exposes **REST APIs for frontend consumption**
  
**FEATURES :**

  Product CRUD operations,

  Category-based product listing,
  
  Local image handling (no cloud required),
  
  H2 in-memory database with console access,
  
  Preloaded sample data,
  
  Clean layered architecture,
  
  RESTful API design,
  
**TECH STACK :**

  Java 17+
  
  Spring Boot
  
  Spring Data JPA
  
  Hibernate
  
  H2 Database
  
  Maven
  
  Lombok
  
**Database Configuration :**

  Uses H2 in-memory database
  
  Database is automatically initialized on application startup
  
  Sample data is inserted using data.sql
